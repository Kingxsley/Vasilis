1. Getting Started

Accessing the Platform

  1. Navigate to your site URL
  2. Click "Login" or "Get Started"
  3. Enter your email and password
  4. Click "Sign In"

User Roles

Role Permissions
Super Admin Full access to all features, manage organizations
Org Admin Manage organization, users, campaigns, content
Media Manager Create and manage content (blog, news, videos)
Trainer View training modules and user progress
User Complete training and view own progress

2. Dashboard Overview

After logging in, you'll see the main dashboard with quick stats, recent activity, and navigation.

Sidebar Navigation

Section Features
Main Dashboard, Organizations, Users
Simulations Campaigns, Ads Tracker, Scenarios
Content CMS, Page Editor, Landing Page, Sidebar, Media Library
Training Modules, Certificates, Templates
Settings Settings, SEO, Email Templates, Security

3. User Management

Adding a New User

  1. Go to Dashboard → Users
  2. Click "Add User" button
  3. Fill in Email, Name, Role, Phone (optional)
  4. Click "Create User"

Importing Users (Bulk)

  1. Click "Import Users"
  2. Download the CSV template
  3. Fill in user data
  4. Upload the completed CSV
  5. Review and confirm

5. Phishing Campaigns

Creating a Campaign

  1. Go to Simulations → Campaigns
  2. Click "New Campaign"
  3. Configure Basic Info (name, type)
  4. Select Target Audience
  5. Choose or create Email Template
  6. Select Landing Page
  7. Schedule or launch immediately

Campaign Metrics

Metric Description
Sent Number of emails delivered
Opened Users who opened the email
Clicked Users who clicked the link
Submitted Users who entered data
Reported Users who reported as phishing

7. Content Management (CMS)

Creating a Blog Post

  1. Go to Content → CMS
  2. Select "Blog" tab
  3. Click "Add New Post"
  4. Fill in Title, Content, Featured Image, Tags
  5. Toggle "Published" to make it live
  6. Click "Save"

Rich Text Editor Features

  • Headings: H1, H2
  • Formatting: Bold, Italic, Underline
  • Lists: Bulleted, Numbered
  • Links: Insert hyperlinks
  • Images: Upload directly or paste URL
  • Quotes: Block quotes
  • Code: Code blocks
Tip: Click the Upload button in the editor to add images directly. Images are automatically optimized!

8. Page Customization

Sidebar Customizer

Customize sidebars for public pages (News, Blog, Videos, About):

  1. Go to Content → Sidebar
  2. Select the page tab
  3. Add sections (CTA, Links, Image, HTML)
  4. Configure each section's content
  5. Reorder using ▲▼ arrows
  6. Click "Save Changes"

Section Types

Type Description
Call to Action Title, description, and button
Quick Links List of navigation links
Image/Banner Display an image
Custom HTML Any custom content

9. Media Library

Uploading Media

  1. Go to Content → Media Library
  2. Click "Upload Media"
  3. Select image(s) from your computer
  4. Images are automatically optimized

Using Media

  1. Find the image you want
  2. Hover and click the Copy icon
  3. Paste the URL wherever needed
Tip: All uploaded images are automatically resized and compressed while maintaining quality. You'll see the % size reduction after upload.

13. Settings & Branding

Branding Settings

Go to Settings → Settings to configure:

Company Info

  • Company Name: Displayed throughout the site
  • Tagline: Shown on landing page

Logo & Favicon

  • Logo: Upload your logo (recommended: 200x200px)
  • Favicon: Browser tab icon (recommended: 32x32px)

Colors

  • Primary Color: Main accent (buttons, links)
  • Text Color: Body text
  • Heading Color: Headings
  • Accent Color: Secondary accent

Footer Settings

  • Custom copyright text
  • Social media links (Facebook, Twitter, LinkedIn, Instagram, YouTube)

14. SEO Settings

Go to Settings → SEO to optimize for search engines:

General Tab

  • Site Title: Browser tab and search results (50-60 chars)
  • Meta Description: Search result description (150-160 chars)
  • Keywords: Comma-separated
  • Canonical URL: Your primary domain

Social Media Tab

  • Open Graph: How your site appears on Facebook/LinkedIn
  • Twitter Card: How your site appears on Twitter

Tracking Tab

  • Google Analytics ID: Enter G-XXXXXXXXXX or UA-XXXXXXXX-X
  • Format is validated automatically
  • Tracking activates when saved

Troubleshooting

Can't Login

  1. Check email spelling
  2. Use "Forgot Password" to reset
  3. Clear browser cache
  4. Try incognito/private mode

Images Not Displaying

  1. Check the image URL is correct
  2. Ensure image is publicly accessible
  3. Try re-uploading to Media Library

Changes Not Appearing

  1. Hard refresh (Ctrl+Shift+R)
  2. Clear browser cache
  3. Wait 30 seconds for cache to update
